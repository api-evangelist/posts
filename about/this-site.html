This website is a static, API-driven toolbox which the API Evangelist uses to make sense of the public API landscape, developing strategies, policies, and rules that can be used to discovery and govern APIs within the enterprise. This website is a federated mix of 30 separate GitHub repositories, APIs, and static websites which then surveys and assesses thousands of other APIs, distilling their APIs and operations down into individual repositories that can be forked and used anywhere.

Each of the subdomains within the API Evangelist website contains information focused on the surveying and assessment of HTTP APIs at scale, with the knowledge of how to do it sold as knowledgebases and services that any company, organization, institution, or government agency can purchase. This is version 4.0 of the API Evangelist platform, with the first version launched in 2010, and this one most notably being 100% out in the open, demonstrating how anyone can survey and assess the API landscape within any industry or enterprise.
