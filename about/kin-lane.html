Kin Lane is the founder and operator of API Evangelist. Kin has been in the technology industry for 35 years with the last 15 years dedicated to HTTP APIs. During the fifteen years operating API Evangelist Kin has been a Presidential Innovation Fellow for the Obama administration, worked with the European Commission on API standards, was Chief Evangelist for Postman, and more recently stood up an API governance program for Bloomberg. Via API Evangelist, Kin is distilling down 15 years of API experience into informative and actionable knowledge bases and services that can help enterprise organizations make sense of the API sprawl that has accumulated throughout this century.

API Evangelist is built upon years of Kin Lane working with companies, organizations, institutions, and government agencies on API strategy. The API Evangelist approach to APIs is not driven by profit motivation, but a strong passion for understanding the technology, business, and politics of doing APIs at scale. API Evangelist generates revenue through selling knowledge, artifacts, partnerships, and services, and is not venture backed. API Evangelist is focused on the internal and external relationship that enterprises have with the markets they operate in, and you can trust that API Evangelist will always provide an expert and trusted view of what is happening in any business sector when it comes to APIs.
